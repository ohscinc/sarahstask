Accessibility

Our commitment to digital access for all.

Sarah's T.A.S.K. believes everyone deserves equitable access to information about communication health. We've worked to make this website accessible to people of all abilities, and we're committed to continuing that effort over time.

Our standards

This website has been designed and developed to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA — the international standard for accessible web content published by the World Wide Web Consortium (W3C). These guidelines explain how to make web content more accessible to people with a wide range of disabilities including visual, auditory, physical, speech, cognitive, language, learning, and neurological differences.

Accessibility features built into this site

We've incorporated specific features to support accessibility, including:

  • Keyboard navigation: Every interactive element on the site can be reached and activated using only a keyboard
  • Skip-to-content links: Keyboard users can jump past the navigation directly to the main content of each page
  • Screen reader support: Pages use proper semantic HTML, ARIA labels, and landmark regions so assistive technology can navigate effectively
  • Visible focus indicators: A clearly visible outline shows keyboard users which element is currently focused
  • Descriptive image alternatives: All meaningful images include alt text describing their content
  • Reduced motion support: Animations are disabled for users with the "Reduce Motion" preference enabled in their operating system
  • Sufficient color contrast: Text and interactive elements meet or exceed the WCAG AA contrast requirements
  • Responsive design: The site adapts gracefully across phones, tablets, and desktops, supporting browser zoom up to 200%
  • Clear, plain-language content: Where possible, we use everyday language and explain clinical terms
  • Multiple ways to contact us: Phone, email, online booking, and intake forms are all available

Known limitations

While we strive for full accessibility, we acknowledge some areas may have limitations:

  • Third-party booking services (e.g., Jane App at sarah-bipath.clientsecure.me) are governed by their own accessibility practices outside our direct control
  • Older browsers and assistive technologies may experience reduced functionality
  • If you encounter any barriers, we want to hear about it (see "Feedback" below)

Need help, or noticed a barrier?

If you're having trouble accessing any content on this site — or if you spot something that could work better for you — please let us know. We'll do our best to provide the information you need in a format that works for you, and we use feedback like yours to keep improving.

Alternative formats

If you need information from this website in an alternative format — large print, plain text, audio, or another format — please reach out using the contact methods above and we'll work with you to make it available.

Ongoing improvement

Web accessibility is an ongoing effort, not a one-time project. We periodically review our content and code to identify and fix accessibility issues, and we incorporate accessibility into the design of any new pages or features.
